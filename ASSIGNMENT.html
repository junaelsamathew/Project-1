sort

Definition: The sort command is used to sort lines in text files or output in a specific order, such as alphabetically or numerically. It's frequently used for organizing data.

Usage: sort [options] [file]

Example 1: Sort a text file in ascending order.

sort file.txt

Example 2: Sort numerically and in ascending order.

sort -n numbers.txt

The sort command is very useful for organizing large datasets efficiently.

awk

Definition: awk is a powerful text-processing tool that allows users to scan and process patterns in text, particularly useful for handling and manipulating columns in text files.

Usage: awk 'pattern {action}' file

Example 1: Print the first column of a file.

awk '{print $1}' file.txt

Example 2: Print columns 1 and 3 of a file.

awk '{print $1, $3}' data.txt

awk is widely used in data analysis and reports generation by extracting specific columns from files.

sed

Definition: The sed (stream editor) command is used for modifying and transforming text streams in files or inputs. It is particularly useful for search-and-replace operations.

Usage: sed 's/pattern/replacement/' file

Example 1: Replace "apple" with "banana" in a file.

sed 's/apple/banana/' fruit.txt

Example 2: Replace text globally within the file.

sed 's/apple/banana/g' fruit.txt

sed can also be used for batch editing files, automating text changes.

cut

Definition: The cut command is used to extract sections from each line of a file or stream. It's helpful for working with structured text like CSV or tab-delimited data.

Usage: cut [options] file

Example 1: Extract the first field from a CSV file.

cut -d',' -f1 file.csv

Example 2: Extract the first and second fields from a CSV file.

cut -d',' -f1,2 data.csv

The cut command simplifies data extraction from structured files.

tr

Definition: The tr command is used to translate or delete characters in the input text, typically useful for character-based transformations.

Usage: tr [set1] [set2]

Example 1: Convert lowercase letters to uppercase.

echo "hello" | tr 'a-z' 'A-Z'

Example 2: Delete specific characters (remove digits from text).

echo "hello123" | tr -d '0-9'

tr is useful for simple text transformations such as case conversion and character deletion.

echo

Definition: The echo command is used to print text or variables to the standard output, often used in scripting for displaying messages.

Usage: echo [options] [string]

Example 1: Display a simple message.

echo "Hello, Linux!"

Example 2: Display the value of an environment variable.

echo $PATH

echo is one of the most commonly used commands in shell scripting.

export

Definition: The export command is used to set environment variables or to mark variables for export to other programs or shell sessions.

Usage: export VAR=value

Example 1: Set a custom environment variable.

export MY_VAR="Hello"

Example 2: Add a directory to the system's PATH variable.

export PATH=$PATH:/new/path

export is essential for modifying system environment variables, especially in scripting.

history

Definition: The history command displays the list of commands that were previously executed in the terminal session.

Usage: history

Example 1: View the command history.

history

Example 2: Search for a specific command in the history.

history | grep "sort"

history is useful for reviewing past commands and re-executing them.

rsync

Definition: The rsync command is used to synchronize files and directories between two locations, often used for backups and file transfers.

Usage: rsync [options] source destination

Example 1: Sync files between directories.

rsync -av source/ destination/

Example 2: Perform a backup operation.

rsync -av /home/user/ /backup/

rsync is a highly efficient command for remote and local file synchronization.

screen

Definition: The screen command allows users to manage multiple terminal sessions within a single window, ideal for long-running processes.

Usage: screen

Example 1: Start a new screen session.

screen

Example 2: Reattach to an existing screen session.

screen -r

screen is invaluable for managing multiple tasks and processes in the background.

uniq

Definition: The uniq command is used to filter out repeated lines in a file or stream. It's typically used to remove duplicates after sorting a file.

Usage: uniq [options] file

Example 1: Remove duplicate lines from a sorted file.

uniq sorted.txt

Example 2: Display only duplicate lines.

uniq -d file.txt

uniq is useful for removing redundancy in files or streams.
